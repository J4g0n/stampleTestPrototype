# Stample Test Prototype
========================

Small prototype to test stample app

## Launch test with chromedriver (dev mode)
 
On mac, open terminal and do: 

0. Install chromedriver:
  > brew install chromedriver

1. Go to this repo:  
  > cd \<PATH_TO_APP\> 

2. Launch chromedriver:  
  > nohup chromedriver \&
  
3. Launch tests:
  > ./activator test
